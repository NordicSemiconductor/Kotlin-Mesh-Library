GenericLevelSet

class GenericLevelSet(var tid: UByte?, val level: Short, val transitionParams: TransitionParameters? = null) : AcknowledgedMeshMessage, TransactionMessage, TransitionMessage

This message is used to set the current status of a GenericLevelServer model. Response received to the message is GenericLevelStatus.
