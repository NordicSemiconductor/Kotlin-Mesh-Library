ModelEventHandler

abstract class ModelEventHandler

Defines the functionality of a Model on the Local Node.

This event handler is assigned to the models when setting up the MeshNetworkManager.localElements.

The event handler must declare a map of mesh message type supported by this Model. Whenever a message matching any of the declared op codes is received, and the model is bound to an Application Key used to encrypt the message. Upon receiving a message, the handle with the ModelEvent will be invoked.
